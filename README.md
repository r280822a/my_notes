# My Notes
Simple notes app made in flutter, intended for android

### Database
Makes use of a SQFLite to store notes on the device

## Features
- Stores notes locally
- Swap 2 notes
- Supports unicode
- Haptic feedback when first selecting a note
- Drag & reorder notes
